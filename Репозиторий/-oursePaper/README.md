# Игра Гранди (КЛИГ)

**Классическая логическая игра для двух игроков с выигрышной стратегией компьютера.**

## О проекте

**Игра Гранди** — это стратегическая математическая игра, относящаяся к классу беспристрастных игр. Проект реализован в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения» (Орский гуманитарно-технологический институт, специальность 09.02.07 Информационные системы и программирование).

 **Особенности:**
- Полностью графический интерфейс (Windows Forms)
- Игрок против компьютера (AI на основе теории Шпрага-Гранди)
- Автоматический поиск выигрышных ходов
- Подробное логирование каждого хода
- Проверка всех правил (запрет на деление кучи на равные части)

---

## 🎮 Правила игры

1. **Начальная позиция** — одна куча из N предметов (камней, спичек).
2. За один ход игрок **обязан** выбрать любую кучу размером **больше 2** и разделить её на **две непустые кучи разных размеров**.
   - Разрешенный ход: `5 → 2 + 3`
   - Запрещенный ход: `4 → 2 + 2` (части равны)
3. Игра заканчивается, когда все кучи имеют размер **1 или 2** (их нельзя разделить по правилам).
4. **Победитель** — тот, кто сделал **последний ход**.

## Установка и запуск

### Системные требования
- **ОС:** Windows 10 / 11
- **ОЗУ:** от 2 ГБ
- **Дисковое пространство:** от 100 МБ
- **Платформа:** .NET Framework 4.7.2 или выше

### Установка
1. Скачайте последнюю версию из раздела [Releases](https://github.com/your-repo/grandi-game/releases)
2. Распакуйте архив в любую папку
3. Запустите `GrandiGame.exe`
   > *Ярлык на рабочем столе создается автоматически (опционально)*
